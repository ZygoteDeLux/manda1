This is an overview of week 1

Can work with Git in the terminal.

use the the following command to initialize a new repository:
git init
now link to a remote repository you want to push your changes to.
git remote add origin [URL]
after initializing git, you stage all changes to git typing the command:
git add -A
you can now commit the staged files with the command:
git commit -m "commit message here"
The -u flag sets the upstream, linking your local branch to the corresponding remote branch.
git push -u origin [branch_name]
After this initial setup, you can use
git push
to push future commits to the same remote branch.

Understand and can work with data types in Javascript.
Understands type coercion and strategies to avoid it.

There are 7 primitive datatypes in javascript and 3 non primitive.

Primitive datatypes:
"Number" represents both integers and float points.
let x = 42;
let y = 3.14;
"String" represents a sequence of characters.
let str = "Hello Anders!";
"Boolean" is a binary variable which is either true or false.
let thisIsTheWay = true;
"Undefined" represents a variable that has been declared but has no value.
let z;
"Null" represents a variable that has been declared and intentionally has no value.
let empty = null;
"Symbols" represents a variable that is garaunteed to be unique.

"BigInt" represents large integers that cant be represented by a number.
const bigInt = 1234567890123456789012345678901234567890n;

"Object" represents a collection of key:value pairs.
